Interactive figure · post-hoc activation steering

A concept isn't alwaysthe loudest direction.

Standard steering computes one vector and adds it everywhere. But the target structure can sit beneath a higher-variance nuisance axis — position, length, formatting. This page builds a tiny synthetic activation set for the word "bank" and lets you watch naive clustering chase the nuisance, then watch suppression recover the two senses and turn them into a steering vector. Everything below is computed live from the points; nothing is canned.

01 / THE TOY ACTIVATION SET

Two senses, one nuisance

Each point is a fake d=2 activation at the "bank" token. Two ground-truth senses are separated by a fixed, modest gap along a hidden semantic axis. A second orthogonal nuisance axis carries token-position / sentence-length variation — and we can crank its variance with the slider in the next panel. A per-point token_position value is generated correlated with the nuisance coordinate, so the detector has metadata to lean on.

02 / DIAGNOSE THE GEOMETRY

Run the variance hierarchy. Ask what PC0 actually is.

PCA gives the variance-ranked axes. The method's first question isn't "what are the clusters" but "is the top axis the signal or a nuisance?" Drag the nuisance slider and watch the regime flip: at low nuisance the top axis is the sense (Dominant); past the crossover the top axis becomes position and naive k-means collapses to chance (Masked).

03 / SUPPRESS, RECOVER, BUILD THE VECTOR

Lift PC0 off. Cluster the residual. Diff the centroids.

When PC0 is flagged, project it out: H_clean = H − H·v₀v₀ᵀ. The cloud collapses onto the residual line where the senses are now the only structure left. Re-cluster, then the steering vector is just the centroid difference, d = mean(C_fin) − mean(C_river) — cleaned of the nuisance it never should have contained.

Residual space · after suppressing PC0—

Purity: what suppression bought youraw → residual
In the Dominant regime, suppression hurts (you deleted the signal). In the Masked regime it rescues the senses. The method only suppresses when the flag fires.
When on, one global vector under-serves both modes — the inference router below uses a local vector per region.
04 / STEER AT INFERENCE

Route the live activation, then push it.

At generation time a "bank" activation arrives. The router scores region membership p_i = softmax(−β‖h−c_i‖²), picks the nearest region's local steering vector, and applies h' = h + α·d. Drag the query; raise α; watch the predicted sense flip from river to financial as it crosses the boundary.

05 / THE FOUR REGIMES

One diagnostic, four interventions

The slider you played with traces the top edge of this table. The whole point of the method is that it diagnoses before it steers — the geometry tells you which tool to reach for.
